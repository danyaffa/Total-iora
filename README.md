# AuraCode

**Sanctuary of reflection.** Not science. Not religion. A mirror.

## Quick Start
1) Copy files to a new repo (GitHub).
2) Create a project on Vercel and connect the repo.
3) Set Environment Variables from `.env.example`.
4) Add Stripe Checkout Links (monthly & lifetime).
5) Add Firebase Web App credentials.
6) Add `OPENAI_API_KEY`.
7) Deploy. Done.

## Notes
- Voice: Uses Web Speech API for input & Speech Synthesis for output.
- Data: Aura Wall notes go to Firestore.
- Safety: We never promise outcomes. All reflections are symbolic.
